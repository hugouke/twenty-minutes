## Description

API GraphQL com NestJS criada em vinte minutos.
Acompanhe o processo de desenvolvimento no [youtube](https://www.youtube.com/channel/UCJ-gtqHhTUXIxaNAMiNshOg).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
